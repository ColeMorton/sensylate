## Parameters

### Core Parameters
- `action`: Documentation action - `analyze` | `categorize` | `consolidate` | `restructure` | `full_optimization` | `help` (required)
- `target_directory`: Documentation directory path (optional, defaults to "./docs")
- `consolidation_level`: Aggressiveness level - `conservative` | `moderate` | `aggressive` (optional, default: moderate)
- `output_format`: Result format - `summary` | `detailed` | `implementation_plan` (optional, default: detailed)

### Advanced Parameters
- `preserve_history`: Maintain change tracking - `true` | `false` (optional, default: true)
- `audience_filter`: Target audience focus - `all` | `developers` | `users` | `technical` (optional, default: all)
- `quality_threshold`: Minimum document quality standard - `basic` | `professional` | `institutional` (optional, default: professional)
- `cross_reference`: Maintain document relationships - `true` | `false` (optional, default: true)
- `backup_originals`: Create backups before changes - `true` | `false` (optional, default: true)

### Optimization Parameters
- `merge_strategy`: How to handle conflicts - `preserve_all` | `best_content` | `most_recent` (optional, default: best_content)
- `format_standardization`: Apply consistent formatting - `true` | `false` (optional, default: true)
- `navigation_enhancement`: Improve document discovery - `true` | `false` (optional, default: true)
- `metadata_enrichment`: Add document metadata - `true` | `false` (optional, default: true)

4. **Redundancy Assessment**: Identify overlapping and duplicate content

- Quality scoring based on institutional standards

## Documentation Assessment Criteria

### Quality Evaluation Framework
**Content Quality Assessment**:
- **Accuracy**: Technical correctness and up-to-date information
- **Completeness**: Comprehensive coverage of topic areas
- **Clarity**: Readability and comprehension for target audience
- **Consistency**: Standardized formatting, style, and terminology
- **Usefulness**: Practical value for intended users

**Organizational Quality Assessment**:
- **Discoverability**: Ease of finding relevant information
- **Navigation**: Clear pathways between related documents
- **Hierarchy**: Logical organization supporting user needs
- **Cross-References**: Effective linking and relationship management
- **Maintainability**: Sustainable structure for ongoing updates

### Consolidation Strategy Framework
**Document Merge Criteria**:
- **Topic Overlap**: Percentage of shared subject matter (>60% = consolidation candidate)
- **Audience Alignment**: Same target user base and expertise level
- **Purpose Similarity**: Compatible document functions and goals
- **Content Quality**: Higher quality content takes precedence
- **Recency**: More recent information preferred when conflicting

**Preservation Priorities**:
1. **Technical Accuracy**: Never compromise factual correctness
2. **Unique Insights**: Preserve all non-redundant information
3. **Historical Context**: Maintain important background information
4. **User Experience**: Prioritize usability and comprehension
5. **Cross-References**: Preserve all valid document relationships
