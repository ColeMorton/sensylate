Usage Instructions

For Developers:

  1. Copy the CSS specifications from STYLE_GUIDE.md
  2. Include Paytone One font: @import url('https://fonts.googleapis.com/css2?family=Paytone+One:wght@400&display=swap');
  3. Apply .personal-brand-complete or .attribution-brand-complete classes
  4. Ensure proper container background (dark) for visibility
  5. Test responsive behavior on mobile devices

For Designers:

  • Personal Brand: Use for navigation, UI elements, direct user interactions
  • Attribution Brand: Use for exported content, social media, charts
  • Both maintain identical styling but different text content and width
  • Preserve the 5px upward transform for visual alignment
